### Projetos em Destaque

#### [Processador de Webhooks Idempotente](https://github.com/lcsqueiroz/processador-de-webhooks-idempotente)
*   **Tecnologias:** Node.js, Redis, Express.js
*   Mecanismo de consumo de webhooks focado em resiliência, utilizando chaves de idempotência armazenadas em memória cache distribuída para evitar o processamento duplicado de requisições idênticas em cenários de retry de rede.

#### [Performance e Análise de Dados via API](https://github.com/lcsqueiroz/performance-e-analise-de-dados-via-api)
*   **Tecnologias:** Node.js, SQL Server/PostgreSQL, Express.js
*   API voltada para processamento de alto volume de dados, implementando otimização de queries, paginação eficiente e estratégias de indexação para mitigar gargalos de I/O em relatórios e dashboards analíticos.

#### [Outbox Pattern com Fila de Eventos](https://github.com/lcsqueiroz/outbox-pattern)
*   **Tecnologias:** Node.js, PostgreSQL, BullMQ (Redis)
*   Implementação do padrão arquitetural Outbox para garantir a consistência eventual e a entrega confiável de mensagens entre serviços, mitigando perdas ou duplicidades em cenários de falha na rede ou indisponibilidade do broker.
